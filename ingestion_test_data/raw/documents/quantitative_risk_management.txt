Quantitative Risk Management in Financial Trading Systems
========================================================

## Introduzione al Quantitative Risk Management

Il Quantitative Risk Management rappresenta l'applicazione di metodi matematici,
statistici e computazionali per identificare, misurare e controllare i rischi finanziari
in sistemi di trading automatizzati. Questa disciplina combina teoria finanziaria,
statistica avanzata e informatica per creare framework robusti di gestione del rischio.

### Evoluzione Storica

Il quantitative risk management ha avuto origine negli anni '90 con l'introduzione
del Value at Risk (VaR) da parte di J.P. Morgan. Da allora, si è evoluto attraverso
diversi paradigmi:

1. **Era VaR (1990s)**: Focus su misure di rischio unidimensionali
2. **Era Coherent Risk Measures (2000s)**: Expected Shortfall e altre misure coerenti
3. **Era Big Data (2010s)**: Machine learning e high-frequency risk management
4. **Era AI-Driven (2020s)**: Sistemi adattivi e predittivi

## Framework Matematici per la Gestione del Rischio

### Teoria dei Rischi Coerenti

La teoria dei rischi coerenti, sviluppata da Artzner et al. (1999), definisce
proprietà che una misura di rischio deve soddisfare per essere considerata "coerente":

#### Proprietà Fondamentali

1. **Monotonicità**: ρ(X) ≤ ρ(Y) se X ≤ Y per ogni stato della natura
2. **Subadditività**: ρ(X+Y) ≤ ρ(X) + ρ(Y)
3. **Omogeneità positiva**: ρ(λX) = λρ(X) per λ > 0
4. **Traslazione**: ρ(X + c) = ρ(X) - c per costante c

#### Misure di Rischio Coerenti

- **Value at Risk (VaR)**: NON coerente (non subadditivo)
- **Expected Shortfall (ES)**: Coerente
- **Entropic Risk Measure**: Coerente
- **Spectral Risk Measures**: Coerente

### Modelli Stocastici per i Rischio di Mercato

#### Modello di Black-Scholes-Merton

Il modello BSM fornisce il framework per il pricing di opzioni e la misura del rischio
in derivati:

```
dS = μS dt + σS dW
```

Dove:
- S: prezzo dell'asset
- μ: rendimento atteso (drift)
- σ: volatilità
- dW: incremento Wiener (rumore browniano)

#### Modelli GARCH per la Volatilità

I modelli GARCH catturano la volatilità cluster e l'effetto leverage:

**GARCH(1,1)**:
```
σ_t² = ω + α ε_{t-1}² + β σ_{t-1}²
```

**EGARCH (Exponential GARCH)**:
```
log(σ_t²) = ω + α |ε_{t-1}| + γ ε_{t-1} + β log(σ_{t-1}²)
```

#### Modelli a Salti (Jump-Diffusion)

Per catturare eventi estremi:

```
dS = μS dt + σS dW + J dN
```

Dove J è la dimensione del salto e N è un processo di Poisson.

```python
import numpy as np
from numba import jit
import pandas as pd

class RiskCalculator:
    def __init__(self, confidence_level=0.99):
        self.confidence_level = confidence_level

    @jit(nopython=True)
    def calculate_var_parametric(self, returns, portfolio_value):
        """
        Calcolo VaR parametrico ottimizzato con Numba
        """
        mean_ret = np.mean(returns)
        std_ret = np.std(returns)
        z_score = 2.3263478740408408  # 99% confidence

        var_return = -(mean_ret + z_score * std_ret)
        return var_return * portfolio_value

    def calculate_portfolio_var(self, positions, covariance_matrix):
        """
        VaR portafoglio con matrice covarianza
        """
        weights = np.array([pos['weight'] for pos in positions])
        volatilities = np.array([pos['volatility'] for pos in positions])

        # Matrice covarianza
        cov_matrix = np.outer(volatilities, volatilities) * covariance_matrix

        # Volatilità portafoglio
        portfolio_vol = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))

        # VaR semplificato (senza drift)
        z_score = 2.3263478740408408
        portfolio_value = sum(pos['value'] for pos in positions)

        return z_score * portfolio_vol * portfolio_value

    def stress_test_portfolio(self, positions, scenarios):
        """
        Stress testing multi-scenario
        """
        results = {}
        portfolio_value = sum(pos['value'] for pos in positions)

        for scenario_name, shocks in scenarios.items():
            portfolio_pnl = 0

            for pos in positions:
                asset_shock = shocks.get(pos['asset'], 0)
                pnl = pos['value'] * asset_shock
                portfolio_pnl += pnl

            results[scenario_name] = {
                'pnl': portfolio_pnl,
                'return_pct': portfolio_pnl / portfolio_value * 100
            }

        return results
```

```python
class LimitManager:
    def __init__(self, initial_limits):
        self.limits = initial_limits
        self.breaches = []

    def check_limits(self, current_positions, market_data):
        """
        Verifica compliance con limiti di rischio
        """
        breaches = []

        # Calcolo metriche rischio correnti
        portfolio_var = self.calculate_portfolio_var(current_positions)
        concentration = self.calculate_concentration(current_positions)
        liquidity_ratio = self.calculate_liquidity_ratio(current_positions, market_data)

        # Verifica limiti
        if portfolio_var > self.limits['max_var']:
            breaches.append({
                'type': 'VaR',
                'current': portfolio_var,
                'limit': self.limits['max_var'],
                'severity': 'HIGH'
            })

        if concentration > self.limits['max_concentration']:
            breaches.append({
                'type': 'Concentration',
                'current': concentration,
                'limit': self.limits['max_concentration'],
                'severity': 'MEDIUM'
            })

        if liquidity_ratio < self.limits['min_liquidity']:
            breaches.append({
                'type': 'Liquidity',
                'current': liquidity_ratio,
                'limit': self.limits['min_liquidity'],
                'severity': 'CRITICAL'
            })

        self.breaches.extend(breaches)
        return breaches

    def adjust_limits(self, market_volatility, portfolio_size):
        """
        Ajustamento dinamico dei limiti basato su condizioni di mercato
        """
        vol_multiplier = min(max(market_volatility / 0.20, 0.5), 2.0)
        size_multiplier = min(max(portfolio_size / 1000000, 0.8), 1.5)

        adjusted_limits = {}
        for limit_name, base_limit in self.limits.items():
            adjusted_limits[limit_name] = base_limit * vol_multiplier * size_multiplier

        return adjusted_limits
```

#### Pre-computed Risk Metrics

Utilizzo di tecniche di pre-calcolo per ridurre latenza:

```python
import numpy as np
from concurrent.futures import ThreadPoolExecutor

class PrecomputedRiskEngine:
    def __init__(self, positions, market_data):
        self.positions = positions
        self.market_data = market_data
        self.precomputed_var = {}
        self.executor = ThreadPoolExecutor(max_workers=4)

    def precompute_risk_surface(self):
        """
        Pre-calcola superficie di rischio per diversi scenari
        """
        price_shocks = np.linspace(-0.05, 0.05, 21)  # -5% to +5%
        vol_shocks = np.linspace(-0.5, 0.5, 11)      # -50% to +50%

        futures = []
        for price_shock in price_shocks:
            for vol_shock in vol_shocks:
                future = self.executor.submit(
                    self.compute_scenario_risk,
                    price_shock,
                    vol_shock
                )
                futures.append(future)

        # Raccogli risultati
        for future in futures:
            result = future.result()
            self.precomputed_var[(result['price_shock'], result['vol_shock'])] = result['var']

    def compute_scenario_risk(self, price_shock, vol_shock):
        """
        Calcola VaR per uno specifico scenario
        """
        # Applica shock ai prezzi e volatilità
        shocked_prices = {asset: price * (1 + price_shock)
                         for asset, price in self.market_data['prices'].items()}

        shocked_vols = {asset: vol * (1 + vol_shock)
                       for asset, vol in self.market_data['volatilities'].items()}

        # Calcola VaR scenario
        portfolio_var = self.calculate_portfolio_var(
            self.positions,
            shocked_prices,
            shocked_vols
        )

        return {
            'price_shock': price_shock,
            'vol_shock': vol_shock,
            'var': portfolio_var
        }

    def get_real_time_var(self, current_market_conditions):
        """
        Interpolazione per ottenere VaR in tempo reale
        """
        price_shock = current_market_conditions['price_change']
        vol_shock = current_market_conditions['vol_change']

        # Interpolazione bilineare
        return self.bilinear_interpolation(price_shock, vol_shock)
```

```python
import tensorflow as tf
from tensorflow import keras
import numpy as np

class VolatilityPredictor:
    def __init__(self, sequence_length=60):
        self.sequence_length = sequence_length
        self.model = self.build_model()

    def build_model(self):
        """
        Costruzione LSTM per previsione volatilità
        """
        model = keras.Sequential([
            keras.layers.LSTM(64, input_shape=(self.sequence_length, 5),
                            return_sequences=True),
            keras.layers.Dropout(0.2),
            keras.layers.LSTM(32),
            keras.layers.Dropout(0.2),
            keras.layers.Dense(16, activation='relu'),
            keras.layers.Dense(1, activation='linear')
        ])

        model.compile(optimizer='adam',
                     loss='mse',
                     metrics=['mae'])

        return model

    def prepare_data(self, price_data, technical_indicators):
        """
        Preparazione dati per training
        """
        # Calcola rendimenti
        returns = np.log(price_data / price_data.shift(1)).dropna()

        # Calcola volatilità rolling
        volatility = returns.rolling(window=20).std()

        # Combina con indicatori tecnici
        features = pd.concat([
            returns,
            volatility,
            technical_indicators['rsi'],
            technical_indicators['macd'],
            technical_indicators['bb_width']
        ], axis=1).dropna()

        # Crea sequenze
        X, y = [], []
        for i in range(len(features) - self.sequence_length):
            X.append(features.iloc[i:i+self.sequence_length].values)
            y.append(volatility.iloc[i+self.sequence_length])

        return np.array(X), np.array(y)

    def train(self, X_train, y_train, epochs=100):
        """
        Training del modello
        """
        early_stopping = keras.callbacks.EarlyStopping(
            monitor='val_loss',
            patience=10,
            restore_best_weights=True
        )

        history = self.model.fit(
            X_train, y_train,
            epochs=epochs,
            batch_size=32,
            validation_split=0.2,
            callbacks=[early_stopping],
            verbose=1
        )

        return history

    def predict_volatility(self, recent_data):
        """
        Previsione volatilità futura
        """
        # Prepara input
        input_sequence = recent_data.values.reshape(1, self.sequence_length, -1)

        # Predizione
        predicted_vol = self.model.predict(input_sequence)[0][0]

        return predicted_vol

    def update_risk_limits(self, predicted_volatility, current_limits):
        """
        Ajustamento limiti basato su volatilità prevista
        """
        vol_multiplier = predicted_volatility / 0.02  # Volatilità base 2%

        adjusted_limits = {}
        for limit_name, base_limit in current_limits.items():
            if 'var' in limit_name.lower():
                adjusted_limits[limit_name] = base_limit * vol_multiplier
            elif 'position' in limit_name.lower():
                adjusted_limits[limit_name] = base_limit / vol_multiplier
            else:
                adjusted_limits[limit_name] = base_limit

        return adjusted_limits
```

```python
import pandas as pd
from datetime import datetime
import xml.etree.ElementTree as ET

class RegulatoryReporter:
    def __init__(self, institution_id, reporting_date):
        self.institution_id = institution_id
        self.reporting_date = reporting_date

    def generate_basel_report(self, risk_metrics, positions):
        """
        Genera report Basel III compliant
        """
        root = ET.Element("BaselIII_Report")
        root.set("institution", self.institution_id)
        root.set("date", self.reporting_date.strftime("%Y-%m-%d"))

        # Market Risk Section
        market_risk = ET.SubElement(root, "MarketRisk")

        # VaR
        var_element = ET.SubElement(market_risk, "ValueAtRisk")
        var_element.set("confidence", "99")
        var_element.set("horizon", "10")
        var_element.text = f"{risk_metrics['var_99']:.2f}"

        # Stressed VaR
        stressed_var = ET.SubElement(market_risk, "StressedVaR")
        stressed_var.text = f"{risk_metrics['stressed_var']:.2f}"

        # Risk Factors
        risk_factors = ET.SubElement(market_risk, "RiskFactors")
        for factor, value in risk_metrics['risk_factors'].items():
            factor_element = ET.SubElement(risk_factors, "Factor")
            factor_element.set("name", factor)
            factor_element.text = f"{value:.4f}"

        # Positions Summary
        positions_summary = ET.SubElement(root, "PositionsSummary")
        for asset_class, data in positions.items():
            asset_element = ET.SubElement(positions_summary, "AssetClass")
            asset_element.set("type", asset_class)
            asset_element.set("value", f"{data['value']:.2f}")
            asset_element.set("var_contribution", f"{data['var_contrib']:.2f}")

        # Convert to string
        tree = ET.ElementTree(root)
        xml_string = ET.tostring(root, encoding='unicode', method='xml')

        return xml_string

    def generate_stress_test_report(self, stress_results):
        """
        Genera report stress testing
        """
        report_data = []

        for scenario, results in stress_results.items():
            row = {
                'Scenario': scenario,
                'PnL': results['pnl'],
                'Return_Pct': results['return_pct'],
                'VaR_Breach': 'Yes' if abs(results['return_pct']) > 5 else 'No',
                'Capital_Impact': results['pnl'] * 0.08  # 8% capital charge
            }
            report_data.append(row)

        df = pd.DataFrame(report_data)

        # Formattazione report
        report = f"""
STRESS TESTING REPORT
Institution: {self.institution_id}
Date: {self.reporting_date.strftime('%Y-%m-%d')}

{df.to_string(index=False, float_format='%.2f')}

SUMMARY STATISTICS:
- Maximum Loss: ${df['PnL'].min():,.2f}
- Scenarios with VaR Breach: {df['VaR_Breach'].value_counts()['Yes']}
- Average Capital Impact: ${df['Capital_Impact'].mean():,.2f}
"""

        return report
```

## Case Studies e Lezioni Apprese

### Caso Knight Capital (2012)

**Contesto**: Market maker high-frequency perde $440M in 45 minuti.

**Causa Radice**:
- Errore nel deployment di software di routing ordini
- Mancanza di kill switch automatici
- Insufficiente testing di deployment

**Lezioni**:
1. Implementare circuit breakers multi-livello
2. Testing rigoroso di deployment con dati di produzione
3. Risk controls indipendenti dai sistemi di trading

### Caso JPMorgan "London Whale" (2012)

**Contesto**: Trading desk CIO perde $6.2B su posizioni CDS.

**Causa Radice**:
- Modelli di rischio inadeguati per posizioni complesse
- Mancanza di oversight indipendente
- Concentrazione rischio in poche posizioni

**Lezioni**:
1. Modelli di rischio specifici per prodotti complessi
2. Oversight indipendente sui trading desk
3. Limiti di concentrazione e stress testing regolare

### Caso Archegos Capital (2021)

**Contesto**: Family office causa perdite $10B+ a banche investment.

**Causa Radice**:
- Mancanza di monitoraggio margin da parte prime brokers
- Concentrazione rischio in poche posizioni altamente leveraged
- Failure to communicate tra controparti

**Lezioni**:
1. Enhanced due diligence per family offices
2. Real-time margin monitoring
3. Contagion analysis attraverso network di controparti
